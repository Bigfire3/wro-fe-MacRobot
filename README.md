# Future Enginners - MacRobot

Dieses Repository enthält alle wichtigen Dateien zu unserem autonom fahrenden Auto "MacRobot" (stammt von der Fernsehserie MacGyver). Verwendet haben wir den Lego EV3 Mindstorms in Kombination mit der PixyCam V1 für Lego Mindstorms. Das Programm wurde in der Lego eigenen Software entwickelt und läauft auf einem EV3-Brick mit der orginalen Lego Mindstorms EV3 Firmware.
